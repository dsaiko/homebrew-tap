# dsaiko/homebrew-tap

Homebrew tap for tools published from `github.com/dsaiko`. Formulas are auto-generated and pushed on each project's release.

## Usage

```bash
brew tap dsaiko/tap
brew install <formula-name>
```

Or in one line:

```bash
brew install dsaiko/tap/<formula-name>
```

## Available formulas

- [`edookit-mcp`](https://github.com/dsaiko/edookit-mcp) — Unofficial MCP connector for Edookit (Czech school information system)
